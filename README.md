## Problem Description

The Inventory Management System for Retail Stores is designed to manage products, stock availability, billing, and sales records efficiently.
The system helps retail shop owners reduce manual work, maintain accurate inventory records, and improve operational efficiency.

---

## How to Run the Code

1. Download or clone the repository.
2. Open the project in your preferred IDE.
3. Compile and run the source code.
4. Follow the on-screen menu/options to use the system.

Example:

```bash
g++ main.cpp -o inventory
./inventory
```

## Sample Input / Output

### Add Product

Input:

```text
Product ID: 10
Product Name: Soya Sticks
Price: 125
Quantity: 10
```

Output:

```text
Product added successfully!
```

### Sell Product

Input:

```text
Product ID: 10
Quantity to Sell: 6
```

Output:

```text
Sale successful!
WARNING: Low stock for Soya Sticks! Only 1 left.
```

### Export Inventory

Output:

```text
Inventory exported successfully to output.csv
```
